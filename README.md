# ACTX - Movie & Actor Intelligence Engine

**ACTX** is a modular Python library designed to retrieve and process actor metadata and filmography from global databases. Developed with a **System Architect** mindset, it prioritizes clean logic, high performance, and robust data handling.

## 🚀 Features
- **Fast Search**: Optimized lookups for actors and celebrities.
- **Deep Data Retrieval**: Extracts bios, filmography, and birth details.
- **Python 3.14 Ready**: Built to handle the latest Python features and compatibility patches.
- **Clean Architecture**: Decoupled logic for easy scaling and integration into larger AI projects like (EVO.AI).

## 🛠️ Tech Stack
- **Language**: Python 3.14
- **Libraries**: Cinemagoer (patched), BeautifulSoup4, HTTPX.
- **Design Pattern**: Object-Oriented Architecture.
